Frequently Asked Questions

Who maintains this site?

This site is in BETA (meaning its evolving a lot at the moment) and is being run by Sarah Bird as part of the Open Contracting Data Standards project

What's with the random references to packages, django, and other things?

This site was built by adapting Open Comparison. Open Comparison is a software that is used to compare software packages, but had features that we could use to compare Open Contracting datasets. Please do let us know if you see a problem like this and we'll fix it.

Many thanks to the awesome Danny and Audrey and other contributors for creating Open Comparison. See more: here

What are grids?

Grids let you compare datasets to each other. A grid comes with a number of default items compared, but you can add more features in order to get a more specific comparison.

What about RSS & Atom Syndication?

The Syndication page has more details.

How can I contribute?

This is an open source project, and we appreciate all sorts of contributions.
